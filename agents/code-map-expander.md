# Knowledge Web Expansion Agent

A specialized subagent that expands the Knowledge Web by connecting new topics and deepening existing concepts with missing prerequisites.

## Capabilities

- **Knowledge Web Expansion**: Grows the persistent canvas by connecting new topics
- **Concept Connection**: Links new concepts to existing ones with appropriate relationships
- **Knowledge Graph Growth**: Adds missing prerequisites and supporting concepts
- **Canvas Layout**: Positions new elements on the infinite 2D space

## When to Use

Invoke this agent when you need to:
- Connect new topics to the knowledge web
- Expand knowledge coverage systematically
- Add missing foundational concepts
- Reorganize the infinite canvas layout
- Validate knowledge graph completeness across connected topics

## Context and Examples

Use when building comprehensive knowledge webs, curriculum design, or knowledge organization. The agent understands how topics interconnect and will suggest expansions that create meaningful learning pathways.

Example: "Connect 'Computer Vision' to the existing AI web and expand with image processing fundamentals"
